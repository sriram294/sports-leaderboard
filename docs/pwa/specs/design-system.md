# Design system

Dark tokens: background `#0A0A0A`, surface `#141414`, lime `#9ADE28`, primary `#F5F5F5`, muted `#9E9E9E`. Use 12–16px radii, 8px spacing rhythm, safe-area padding, lime for active/positive states, and red only for destructive/error states. Shared controls must remain semantic buttons/inputs.
